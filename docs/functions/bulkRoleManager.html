      Add or remove a role from multiple members at once with tracking

      Parameters

      • role: Role

        The role to add or remove

      • members: GuildMember[]

        Array of guild members to modify

      • action: "add" | "remove"

        Whether to 'add' or 'remove' the role

      • Optionaloptions: { continueOnError?: boolean; reason?: string; trackErrors?: boolean }

        Configuration options

        • OptionalcontinueOnError?: boolean

          Whether to continue if an operation fails (default: true)

        • Optionalreason?: string

          Audit log reason for the role changes

        • OptionaltrackErrors?: boolean

      Returns Promise<{ errors?: Error[]; failed: number; success: number }>

      • Results of the operation

      Example

      // Add a role to multiple members
      const inactiveMembers = await getInactiveMembers(guild, 30); // Members inactive for 30+ days
      const results = await bulkRoleManager(
        inactiveRole,
        inactiveMembers,
        'add',
        { reason: 'Marked inactive after 30 days' }
      );

      await channel.send(`Role added to ${results.success} members. Failed: ${results.failed}`);

      // Remove a role from multiple members
      const subscriberRole = guild.roles.cache.find(r => r.name === 'Subscriber');
      const expiredSubscribers = members.filter(m => subscriptionExpired(m));

      const results = await bulkRoleManager(
        subscriberRole,
        expiredSubscribers,
        'remove',
        { reason: 'Subscription expired' }
      );
