      Sends a paginated message with navigation buttons

      Parameters

      • channel: DMChannel | GuildTextBasedChannel

        The channel to send the paginated message to

      • pages: (string | MessageCreateOptions)[]

        Array of message content for each page

      • Optionaloptions: {
            buttons?: ButtonOption[];
            startPage?: number;
            timeout?: number;
            userId?: string;
        }

        Configuration options for the pagination

        • Optionalbuttons?: ButtonOption[]

          Custom buttons to add to the navigation

        • OptionalstartPage?: number

          The initial page index to display (default: 0)

        • Optionaltimeout?: number

          Time in ms before buttons are disabled (default: 60000)

        • OptionaluserId?: string

          If set, only this user can interact with the buttons

      Returns Promise<void>

      • Resolves when the collector ends

      Example

      // Simple text-based pages
      const pages = [
        'Page 1: Introduction to the bot',
        'Page 2: Basic commands',
        'Page 3: Advanced features'
      ];

      // Send paginated message that only the command user can interact with
      await sendPaginatedMessage(interaction.channel, pages, {
        userId: interaction.user.id,
        timeout: 120000 // 2 minutes
      });

      // More complex pages with embeds
      const richPages = [
        {
          embeds: [new EmbedBuilder().setTitle('Commands').setDescription('List of commands')],
          content: 'Page 1'
        },
        {
          embeds: [new EmbedBuilder().setTitle('Settings').setDescription('Bot settings')],
          content: 'Page 2'
        }
      ];

      // With custom buttons
      await sendPaginatedMessage(channel, richPages, {
        buttons: [
          {
            id: 'refresh',
            label: '🔄 Refresh',
            style: 'SECONDARY',
            callback: (page) => page
          }
        ],
        startPage: 1 // Start on second page
      });
